Method

Truth = reviewer-confirmed fields on verified receipts (status reviewed / reconciled / exported / archived). Prediction = extraction_json captured before human edit. Match: amount = numeric equality; date = exact YYYY-MM-DD; currency = case-insensitive; merchant = trimmed case-insensitive exact; expense type = enum equality scored only where truth is set. Headline verdict uses the most recent 3 month-buckets present. A field is flagged when its rolling accuracy is below threshold.
